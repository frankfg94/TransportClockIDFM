# @transport-clock/realtime-vehicles

Optional experimental plugin for estimated realtime vehicle positions. It owns
the client engine, settings UI, VueFlow nodes, CSS/image assets, Nitro route,
IDFM providers, geometry metrics, health check, and legacy settings migration.

Activate it with:

    ENABLED_PLUGINS=@transport-clock/realtime-vehicles

See the [host package guide](../nuxt-plugin-host/README.md) for standalone
installation, authoring, publication, compatibility, and core-only build
instructions. The repository-specific architecture remains in
[docs/plugins.md](../../docs/plugins.md).
